## Stack :hammer:

To make this we use :
- Language : Html, Css, Javascript, Glsl
- Package Manager : Npm 
- Frontend tooling : ViteJs :zap:

## Get the repo :wrench:

- Pull the project 
```
git pull https://github.com/JeremyTar/SolarSysteme.git
```

- Mouve in folder
```
cd solarsysteme
```

- use 
```
npm install
```

- Launch app
```
npm run dev
```

- Explore each planete ! :earth_americas: :ringed_planet: :rocket:
